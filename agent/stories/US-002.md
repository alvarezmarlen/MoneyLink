# USER HISTORY

## ID UH-002

## Título Registro/Inicio de sesión luego de haber seleccionado la tasa de cambio y cantidad a transferir

COMO:     usuario que envía remesas internacionales
QUIERO:  registrarme o iniciar sesión después de seleccionar la tasa y el monto
PARA: poder continuar con el trámite de envío de divisas de forma segura

```gherkin

Escenario 1:  Redirección a registro/inicio de sesión

DADO que el usuario ya seleccionó la tasa de cambio y el monto a transferir
CUANDO presiona el botón para continuar con el envío
ENTONCES el sistema lo redirige a la pantalla de registro o inicio de sesión

Escenario 2: Registro de un nuevo usuario

DADO que el usuario no tiene cuenta
CUANDO completa el formulario de registro con los datos requeridos (nombre, email, contraseña, identificación)
ENTONCES  el sistema valida los datos Y crea una nueva cuenta Y permite continuar con el proceso de envío

Escenario 3: Inicio de sesión de usuario existente

DADO que el usuario ya está registrado
CUANDO introduce su email y contraseña
ENTONCES el sistema valida las credenciales
Y permite continuar con el proceso de envío

Escenario 4: Manejo de errores de autenticación

DADO que el usuario intenta registrarse o iniciar sesión
CUANDO introduce datos incorrectos, incompletos o inválidos
ENTONCES el sistema muestra mensajes claros de error Y no permite avanzar hasta corregirlos

  
```

## Notas

[Esta HU depende de que el usuario ya haya pasado por el conversor de divisas].

- El sistema debe almacenar temporalmente la tasa y el monto seleccionados para continuar el flujo después del login.

- El backend puede ser JSON Server o un mock de autenticación.

- Debe garantizarse la seguridad mínima de contraseñas (validación, longitud, formato).
