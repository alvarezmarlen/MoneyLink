## ID
HU-XXX

## Título
Consulta de hora local en país de destino

## Descripción
**Como** Usuario que envía remesas internacionales
**Quiero** Conocer la hora actual en el país de destino del envío
**Para** Conocer si es horario operativo en país destino.

## Criterios de Aceptación

### Escenario 1: Visualización de la hora al seleccionar país
```gherkin
Dado que el usuario elige un país de destino en el formulario
Cuando el sistema detecta la selección del país
Entonces debe mostrar la hora exacta en formato de 12 horas (AM/PM) indicando claramente el nombre de la zona horaria (ej: "Hora local en Bogotá")
```

### Escenario 2: Cambio dinámico de país
```gherkin
Dado que el sistema ya muestra la hora de un país seleccionado
Cuando el usuario cambia de país o permanece en la pantalla de envío
Entonces el reloj debe actualizarse automáticamente segundo a segundo sin que el usuario tenga que refrescar manualmente la página
```

## Notas
 API: Se usara API externa de zonas horarias.

 UI/UX: El reloj estara cerca del campo País de destino.

 Dependencias: El listado de países debe tener mapeado su correspondiente código de zona horaria (ej: "America/Caracas").
