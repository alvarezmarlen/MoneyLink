# Historia de Usuario

## ID
US-001

## Título
[Convertidor de monedas]

## Descripción
**Como** [usuario que envía remesas internacionales]
**Quiero** [disponer de un convertidor de divisas]
**Para** [evitar hacer cálculos fuera de la aplicación y conocer el valor real de mi transferencia]

## Criterios de Aceptación

### Escenario 1: [Conversión de divisas en tiempo real]
```gherkin
Dado [Que el usuario elige dos tipos de moneda de dos paises diferentes para convertirlo]
Cuando [el usuario introduce un valor numérico en la moneda de origen]
Entonces [la aplicación consulta una API de conversión en tiempo real
Y muestra el valor convertido en la moneda destino con símbolo y bandera correspondientes]
```
### Escenario 2: [Actualización de la tasa cambiaria al hacer clic]
```gherkin
Dado [que el usuario desea conocer la tasa actualizada]
Cuando [hace clic sobre la tasa cambiaria mostrada]
Entonces [ la aplicación consulta nuevamente la API
Y muestra el valor actualizado de la tasa en tiempo real]
```
### Escenario 3: [Continuación del flujo hacia la transferencia]
```gherkin
Dado [que el usuario ya tiene el monto convertido y la tasa aplicada]
Cuando [presiona el botón de ejecución de la transferencia]
Entonces [la aplicación lo redirige a la página de registro/inicio de sesión para continuar el proceso]
```

## Notas
- Los valores de conversión provienen de una API externa.

- Cada país debe mostrar su bandera y símbolo de moneda.

- El botón final confirma el inicio del proceso de envío.

- Debe manejarse el error si la API no responde.

- El monto debe validarse como numérico.
