# Project Context — MoneyLink

MoneyLink es una aplicación para el envío de remesas internacionales. Permite a los usuarios:

- Seleccionar país de origen y destino
- Convertir divisas en tiempo real
- Iniciar registro o sesión para continuar el flujo de envío
- Registrar remitentes y destinatarios
- Autollenar datos en futuros envíos
- Editar y eliminar perfiles guardados
- Realizar seguimiento del envío en tiempo real
- Visualizar tasas cambiarias actualizadas
- Proteger datos sensibles con enmascarado y acceso restringido

El objetivo es ofrecer un flujo rápido, seguro y claro para usuarios que envían dinero a otros países.

## Objetivos del proyecto
- Simplificar el proceso de envío de divisas
- Reducir errores mediante autollenado y validaciones
- Ofrecer transparencia en tasas y montos
- Integrar APIs externas para conversión en tiempo real
- Mantener datos persistentes mediante JSON Server
- Habilitar registro/inicio de sesion para continuar el flujo
- Permitir gestion completa de perfiles (edicion, eliminacion)
- Garantizar seguridad y resguardo de datos personales
- Mostrar seguimiento del envio con actualizacion automatica

## Público objetivo
Usuarios que envían dinero a familiares o contactos en otros países.

## Tecnologías
- Vue 3
- JSON Server
- Fetch API
- Polling simple para actualizaciones
- Tailwind / CSS Glassmorphism
- OpenCode (IA Pair Programming)
